# 茶葉蛋食譜

## 食材

雞蛋 10顆

## 滷汁材料

萬用滷包 1包

紅茶茶包 3包

烏龍茶葉 10克

## 調味料

水 適量

醬油 0.5杯

鹽巴 2匙

冰糖 2匙

八角 2粒

## 作法

1.將雞蛋外殼清洗乾淨

2.用湯匙在鈍端（氣室）敲出裂縫

3.電鍋內鍋 加入雞蛋及略低於雞蛋的水量

4.電鍋內鍋 加入萬用滷包 紅茶茶包

5.電鍋內鍋 加入醬油 鹽巴 冰糖 八角

6.電鍋外鍋 加入2杯水並按下開關

7.開關跳起來後 燜煮30分鐘

8.開關跳起來後 用湯匙將蛋殼均勻敲出裂縫

9.重複步驟6和步驟7

10.於室溫冷卻後放冰箱保存
